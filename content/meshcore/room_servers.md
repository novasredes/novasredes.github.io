---
title: Room servers
description: Mailboxes for store-and-forward messaging
---

We offer several room servers which let you join, write a message and then leave.
These messages are stored on the server such that someone else can login and read
them.

This is, in affect, a mailbox - just a public/global one that all can write to
and read from.
